**项目简介**
Chat 是基于 Linux + C++11 开发的高性能聊天室服务端程序，核心技术栈如下：
核心网络模块：基于 muduo 网络库 开发（Reactor 反应堆模型、多线程高并发设计），muduo 为项目的核心网络通信底层依赖
消息广播核心：集成 Redis 发布订阅 (pub/sub) 实现跨客户端、跨连接的聊天室消息实时广播推送
Redis 客户端驱动：采用 hiredis 客户端库 (编译启用 --with-stream 流式模块) 实现 C++ 与 Redis 的交互
项目构建工具：CMake 工程化管理，跨平台编译友好
项目特性：高性能、高并发、低耦合，模块化设计，支持聊天室频道订阅、消息发布、在线通讯等核心聊天室功能，依托 muduo 的事件驱动模型可支撑大量客户端同时连接。
